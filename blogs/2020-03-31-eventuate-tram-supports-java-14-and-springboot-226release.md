<h1 id="eventuate-tram-supports-java-14-and-springboot-226release-alltestsgreenonjdk14">Eventuate Tram supports Java 14 and @SpringBoot 2.2.6.RELEASE #AllTestsGreenOnJDK14</h1>

<p>We just upgraded the development branches of a couple of example applications to Java 14 and @SpringBoot 2.2.6.RELEASE:</p>

<ul>
  <li>
    <p><a href="https://github.com/eventuate-tram/eventuate-tram-examples-customers-and-orders/tree/development">Eventuate Tram Customers and Orders - Spring Boot</a> - demonstrates how to maintain data consistency in an Spring Boot, JPA-based microservice architecture using <a href="http://microservices.io/patterns/data/saga.html">choreography-based sagas</a>.</p>
  </li>
  <li>
    <p><a href="https://github.com/eventuate-tram/eventuate-tram-sagas-examples-customers-and-orders/tree/development">Eventuate Tram Sagas Customers and Orders - Spring Boot</a> - demonstrates how to maintain data consistency in an Spring Boot, JPA-based microservice architecture using <a href="http://microservices.io/patterns/data/saga.html">orchestration-based sagas</a>.</p>
  </li>
</ul>

<p>When the Eventuate Tram libraries, which are <a href="/post/eventuate/2020/03/31/upcoming-eventuate-release.html">currently <code class="language-plaintext highlighter-rouge">RC</code></a>, are released the changes will be merged into master.</p>
