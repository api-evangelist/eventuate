<h1 id="new-version-of-the-eventuate-platform-released">New version of the Eventuate platform released!</h1>

<p>We are excited to announce that a new version of the Eventuate platform has been released:</p>

<ul>
  <li>Eventuate Tram <a href="https://bintray.com/eventuateio-oss/eventuate-maven-release/eventuate-tram/0.24.0.RELEASE/link"><img src="https://api.bintray.com/packages/eventuateio-oss/eventuate-maven-release/eventuate-tram/images/download.svg?version=0.24.0.RELEASE" /></a></li>
  <li>Eventuate Tram Sagas <a href="https://bintray.com/eventuateio-oss/eventuate-maven-release/eventuate-tram-sagas/0.13.0.RELEASE/link"><img src="https://api.bintray.com/packages/eventuateio-oss/eventuate-maven-release/eventuate-tram-sagas/images/download.svg?version=0.13.0.RELEASE" /></a></li>
  <li>Eventuate Local <a href="https://bintray.com/eventuateio-oss/eventuate-maven-release/eventuate-local/0.32.0.RELEASE/link"><img src="https://api.bintray.com/packages/eventuateio-oss/eventuate-maven-release/eventuate-local/images/download.svg?version=0.32.0.RELEASE" /></a></li>
  <li>Eventuate CDC Service <a href="https://bintray.com/eventuateio-oss/eventuate-maven-release/eventuate-cdc/0.6.1.RELEASE/link"><img src="https://api.bintray.com/packages/eventuateio-oss/eventuate-maven-release/eventuate-cdc/images/download.svg?version=0.6.1.RELEASE" /></a></li>
</ul>

<p>There are several major enhancements to the platform including:</p>

<ul>
  <li>Support for <a href="https://micronaut.io/">Micronaut</a> - see the <a href="/exampleapps.html">Micronaut versions of the Eventuate example apps</a> and the <a href="/docs/manual/eventuate-tram/latest/">user manual</a></li>
  <li>Improved Eventuate CDC service performance - more on that in a later post</li>
  <li><a href="https://start.eventuate.io/">start.eventuate.io</a> - for quickly bootstrapping an Eventuate service: search for the Eventuate dependencies</li>
</ul>

<p>Please read this previous post on <a href="/2020/03/31/upcoming-eventuate-release.html">upgrading to this new version</a>.</p>
