<h1 id="eventuate-now-supports-quarkus">Eventuate now supports Quarkus</h1>

<p>We are very excited to announce that Eventuate now supports the <a href="https://quarkus.io/">Quarkus framework</a> in addition to Spring Boot, Micronaut, and .NET.</p>

<p>You can now easily write Quarkus-based that use the <a href="/post/eventuate/2020/02/24/why-eventuate.html">Saga and CQRS patterns</a>.</p>

<p>Please see the <a href="/exampleapps.html">example applications</a> and the <a href="/gettingstarted.html">getting start docs</a>.</p>
